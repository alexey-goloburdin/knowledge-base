# Все книги таблицей
```dataviewjs
let i = 1;
dv.table(["#", "Имя"], 
  dv.pages()
    .where(p => p["Тип"] && p["Тип"].path === "types/Книга.md")
    .where(p => !["templates", "templater"].includes(p.file.folder))
    .sort(p => p.file.name)
    .map(p => [i++, p.file.link])
)
```
